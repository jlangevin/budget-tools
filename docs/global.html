calculateMonthlyPayment(rate, duration, principal) → {number}

Calculates monthly loan payment

Equation:

P = (Pv*R) / [1 - (1 + R)^(-n)]

P = Monthly payment; Pv = Present value (starting value of the loan); APR = Annual Percentage Rate; R = Periodic interest rate = APR/number of interest periods per year; n = Total number of interest periods (interest periods per year * number of years)

Parameters:
Name Type Description
rate number

Yearly rate (APR)
duration number

Length of loan term in years
principal number

Amount borrowed
